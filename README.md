# klknn mono repository

This repository is a polyglot mono repository by @klknn.

## Contents

- WORKSPACE: Bazel config for third party http/git deps.
- klknn: Root library directory.
  - cc: C++ codes.
  - java: Java codes.
  - python: Python codes.

All the codes must be configured by Bazel.
